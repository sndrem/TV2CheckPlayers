Spillersjekkeren

Spillersjekkeren er en applikasjon som lar deg velge et lag fra Alt Om Fotball og en mappe du har lokalt på maskinen for så å sjekke om du har spillerne som walkin i mappen. Applikasjonen er i testfasen og det kan forekomme feil. Programmet brukes derfor på eget ansvar.

Spillersjekkeren er open source og alle kan være med å bidra med endringer eller komme med innspill.

1. Trykk på .zip og last ned hele mappen.
2. Unzip mappen og legg mappen på ønsket sted du selv velger.
3. Dobbelklikk på application.jar og start programmet.

Hvordan virker det?

Applikasjonen henter automatisk spillernavn fra Alt om Fotball sine lagsider. Lagene er basert på et navn og en ID. ID finner du via altomfotball. Skriv inn navn på lag og les så av teamId=xxx via url'en. Applikasjonen er tenkt brukt til Premier League, men en hvilken som helst ID fra Alt om Fotball vil fungere. Det er også mulig å legge til eller fjerne lag etter eget ønske. Når man legger til et lag blir det automatisk lagret slik at neste gang du åpner applikasjonen så er laget lagret. Det samme gjelder dersom du fjerner et lag. Det skal bare være nødvendig å legge til og slette lag etter endt sesong når lag rykker opp og ned. 

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages
